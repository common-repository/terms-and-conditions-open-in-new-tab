The \"Terms and Conditions\" hyperlink within the WooCommerce checkout page will open in a new tab.

== Description ==
This is a no frills plugin to turn the default WooCommerce \"Terms and Conditions\" hyperlink (located at checkout) from an accordion expand to open the Terms and Conditions url in a new tab. 



== Installation ==
1. Download plugin
2. Upload plugin
3. Activate plugin
You now should have the Terms & Conditions hyperlink open in a new tab upon click (when at checkout of your WooCommerce website).
